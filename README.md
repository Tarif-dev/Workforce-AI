# Workforce AI

AI-powered workforce assistant with:

* NestJS Backend
* Next.js Frontend
* Clerk Authentication
* PostgreSQL (Neon)
* Prisma ORM
* WhatsApp Integration (Planned)
* Slack Integration (Planned)
* Microsoft Teams Integration (Planned)
* Ollama Local LLM Support (Planned)

## Monorepo Structure

apps/
├── api
└── web

## Development

Backend:

cd apps/api
npm run start:dev

Frontend:

cd apps/web
npm run dev
